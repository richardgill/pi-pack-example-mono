## Install with pi-pack

Install `pi-pack` globally:

```bash
npm install -g pi-pack
```


```bash
pi-pack install "git:github.com/richardgill/pi-pack-example-mono" --extension "extension-2"
```


## Add another extension

```bash
cd pi-pack-example-mono/
pi-pack create
```
